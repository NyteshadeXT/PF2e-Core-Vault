# Speak with Coral
## Feat 1
### 

**Category** ancestry; 




You can communicate with any kind of coral, allowing you to find conversation partners in coral reefs of all types and kinds. You can ask questions of, receive answers from, and use the Diplomacy skill with coral. The GM determines which animals count as coral for the purpose of this ability.
