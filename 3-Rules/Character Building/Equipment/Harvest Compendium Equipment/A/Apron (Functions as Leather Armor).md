# Apron (Functions as Leather Armor)

### 

**Category** light; **Group** leather
**Price** 2 gp; 
**Bulk** 1; **Size** med
**Usage** wornarmor

This leather apron functions as a set of leather armor, except it takes 3 Interact actions to don the apron and 1 Interact action to remove it.
