# Miner's Harness

### 

**Category** light; **Group** leather
**Price** 3 gp; 
**Bulk** 1; **Size** med
**Usage** wornarmor

The miner's harness functions as a set of [[Equipment/Leather Armor|Leather Armor]], except it takes 3 Interact actions to don the harness and 1 Interact action to remove it. In addition, you gain a +1 item bonus when [[Actions/Climb|Climbing]] using the harness alongside a [[Equipment/Climbing Kit|Climbing Kit]]
