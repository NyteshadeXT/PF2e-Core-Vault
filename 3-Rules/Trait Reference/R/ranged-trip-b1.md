# Ranged Trip  
*Source: Bestiary p. 85*  

The weapon can be used to [[3-Rules/Character Building/Actions/Trip|Trip]] with the [[Skills#Athletics|Athletics]] skill at a distance up to the weapon's first range increment. The skill check is attempted with a –2 circumstance penalty. You can add the weapon's item bonus to attack rolls as a bonus to the check. As with using a melee weapon to [[3-Rules/Character Building/Actions/Trip|Trip]], a ranged trip weapon doesn't deal any damage when used to [[3-Rules/Character Building/Actions/Trip|Trip]]. This trait usually appears only on a [thrown](3-rules/trait%20reference/t/thrown.md "Thrown Weapon Trait") weapon.
