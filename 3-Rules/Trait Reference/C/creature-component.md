# Creature Component  
*Source: Harvest Compendium p. 8*  

Creature components can be used as, and sometimes be crafted into, unique equipment. Crafting something from creature components follows the standard crafting rules found in the Core Rulebook and always has the component as a craft requirement. The listed price of the creature component is subtracted from the raw material cost needed to craft the item. The GM can switch a specific creature component for another suitable material to better suit their game.

Some components act as consumables that allow you to cast a spell. Unless stated in the item, you do not need to have a spellcasting ability to use the item in this way, provided all other requirements for using the item are met.

Creature components cannot be crafted, only harvested.
